# Student Management System – Advanced SQL Analytics Project

## Project Summary

This project presents a robust relational database system designed to manage and analyze student academic data. It goes beyond basic SQL by incorporating data validation, aggregation, ranking, and performance analysis to generate actionable insights.

The goal is to simulate a real-world academic environment and demonstrate data-driven decision-making using SQL.
## Overview

This project demonstrates the design and analysis of a Student Management System using SQL. It focuses on managing student data, course enrollments, and academic performance while generating insights for decision-making.
## Database Structure

### The system includes:

* Students
* Instructors
* Courses
* Enrollments
* Grades

Built using primary and foreign keys to ensure data integrity.
## Key Analysis
* Retrieved students and their enrolled courses
* Identified top-performing and low-performing students
* Calculated average score per course
* Counted students per course and instructor
* Ranked students using window functions (RANK)
## Key Insights
* High-performing students were identified through ranking
* Some students require academic support (score < 50)
* Course enrollment trends highlight popular subjects
## Tools Used
* SQL (MySQL)
* Relational Database Design
## Conclusion

This project showcases how SQL can be used to design databases, analyze data, and generate actionable insights in an academic setting.
